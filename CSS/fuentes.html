Fuentes

Elegir una fuente apropiada tiene un fuerte impacto en todos los lectores que experimenta un sitio web.

La fuente correcta puede crear una reconocible entidad para una marca o pagina web.

Usar una fuente facil de leer es muy importante. ya que añade valor a un texto. Tambien es importante elegir un correcto color y tamaño.

Familia de fuentes genericas

En CSS hay 5 familias de fuentes genericas

  1. Fuente con serif tienen un pequeño trazo en las puntas de cada letra
  2. Fuentes sans-serif.No tienen los trazos de las enteriores y tiene un aspecto , minimalista y moderno
  3. Fuentes monospace. todas las letras tienen la misma anchura.crean un aspecto mecanico. La fuente que vemos en nuestro codigo es de este tipo.
  4. Fuentes cursivas. Imitan la caligrafia humana
  5. fuentes Fantasy.ÇTienen un aspecto decorativo/jugueton

Tamaño de fuentes

la propiedad font- size especifica el tamaño de un texto.

Ser capaz de modificar los tamaños de los diferentes textos de una web es muy importante.Sin embargo , no uses parrafos con tamaños mayores para que parezcan titulos y viceversa. usda las correspondientes etiquetas h1-h6 para titulos p para parrafos

Los valores de tamaño absolutos o relativos

valor absoluto

  • establece el texto a un tamaño especifico
  • No permite que e l usuario cambie el tamaño de texto desde su ordenador(malo por razones de accesibilidad)
  • Un valor absoluto es util cuando se conoce el tamaño fisico del dispositivo con el que se vizualiza nuestra pagina.

Valor relativo:

  • establece el tamaño relativo a los elementos de alrededor.
  • permite al usuario cambiar el tamaño de texto en el navegador
